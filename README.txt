The project is programmed in python 2.7.

It requires additional toolkits.

NLTK toolkits, for regular expression tokenization, stemming, and stopwords
numpy toolkits, for vector calculation
matplotlib toolkits, for graph plot.

Of course, you can simply setup Spyder which contains Anaconda, which including python IDE spyder and the above apis.

Run program:

Simply, run python Phase1.py or Phase2.py in command line.

Phase1.py is preprocessing the data. Seperate corp into 1400 docs, generate Inverted file.
Phase2.py is build ranked search engine.

Once you run phase1.py, the file will be generated, you don't need to do it again.

For data structure design, please see ReadMe-phase1.doc
